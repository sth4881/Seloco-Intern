OpenCV 함수 정리

cv2.imread(fileName, flag) : fileName은 이미지 파일 경로, flag는 읽어올 때 설정할 옵션(3가지)
cv2.IMREAD_COLOR : 이미지 파일을 Color로 읽음. 투명한 부분은 무시함. 기본값.
cv2.IMREAD_GRAYSCALE : 이미지 파일을 Grayscale로 읽음. 이미지 처리시 중간 단계로 많이 사용됨.
cv2.IMREAD_UNCHAGED : 이미지 파일을 alpha channel까지 포함해 읽음.

image.shape(x, y, z) : 이미지 파일의 모양을 return

cv2.imshow('title', image) : title은 윈도우 창의 제목을 의미, image는 cv2.imread()의 return값
cv2.waitKey(0) : 키보드 입력을 대기하는 함수로 0이면 key 입력이 있을 때까지 무한대기
cv2.destroyAllWindows() : 화면에 나타난 윈도우를 종료

cv2.namedWindow('image', flag) : 윈도우를 고정시킬지, 사용자가 크기를 조절하게 만들 것인지 정함
cv2.WINDOW_NORMAL : 원본 이미지 크기로 윈도우를 생성하되 사용자가 크기를 조절 가능
cv2.WINDOW_AUTOSIZE : 원본 이미지 크기로 고정해서 윈도우를 생성
cv2.imwrite('result.png', original) : 이미지 저장 함수

cap = cv2.VideoCapture(0) : 비디오 캡처 객체를 생성하는 함수. ()안의 숫자는 어떤 카메라를 사용할 것인지 나타내는 장치 인덱스로, 1개만 부착되어 있으면 0, 2개 부착되어 있으면 첫번째는 0, 두번째는 1
cap.isOpened() : 비디오 캡처 객체가 정상적으로 Open되는지 확인
ret, frame = cap.read() : 비디오를 한 프레임씩 읽어옴. 프레임을 정상적으로 읽으면 ret=True, 실패하면 ret=False. frame에 읽은 프레임이 나옴
cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY) : 읽어온 frame을 흑백으로 변환
cap.release() : Open한 캡처 객체를 해제
